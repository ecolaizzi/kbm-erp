# KBM Keyboard Shortcuts Reference

**Versione**: 1.0  
**Data**: 2026-06-09  
**Status**: ✅ Draft

---

## PRINCIPIO

In KBM ogni azione è accessibile da tastiera. Il mouse è uno strumento opzionale, non obbligatorio. Gli utenti ERP esperti operano principalmente con tastiera per massimizzare la produttività.

---

## 1. SHORTCUT GLOBALI (qualsiasi vista)

| Shortcut | Azione | Note |
|---|---|---|
| **F5** | Aggiorna / Refresh | Ricarica dati dalla lista o form corrente |
| **Ctrl+P** | Stampa | Stampa documento o report corrente |
| **Ctrl+F** | Cerca / Focus ricerca | Sposta focus nella searchbox |
| **Alt+H** | Dashboard (Home) | Naviga alla dashboard |
| **Alt+1** | Menu: Core / Admin | Apre primo modulo nel menu |
| **Alt+2** | Menu: Anagrafiche | Apre secondo modulo |
| **Alt+3** | Menu: Listini | Apre terzo modulo |
| **Alt+4..9** | Menu moduli successivi | Naviga ai moduli 4-9 del menu |
| **F1** | Aiuto contestuale | Apre help per la vista corrente |
| **Ctrl+/** | Ricerca globale | Apre search box globale |

## 2. VISTA LISTA / GRIGLIA

| Shortcut | Azione | Note |
|---|---|---|
| **Ctrl+N** | Nuovo record | Apre form nuovo nella vista dettaglio |
| **Enter** | Apri dettaglio | Apre riga selezionata in vista |
| **Ctrl+E** | Modifica selezionato | Apre forma in modalità edit |
| **Delete** | Elimina selezionato | Richiede conferma se record singolo |

| **Ctrl+Shift+F** | Filtro avanzato | Apre pannello filtri avanzati |
| **Ctrl+Shift+C** | Configura colonne | Apre selezione colonne visibili |
| **Ctrl+A** | Seleziona tutto (pagina) | Seleziona tutte le righe della pagina |
| **Shift+Click** | Selezione intervallo | Seleziona righe da A a B |
| **Ctrl+Click** | Selezione multipla | Aggiunge/rimuove dalla selezione |
| **Arrow Up/Down** | Naviga righe | Sposta selezione riga su/giù |
| **Page Up/Down** | Pagina precedente/successiva | Navigazione tra pagine risultati |
| **Ctrl+Home** | Prima riga | Vai alla prima riga |
| **Ctrl+End** | Ultima riga | Vai all'ultima riga della pagina |
| **F4** | Lookup (se campo selezionato) | Apre lookup entità se in inline edit |
| **Esc** | Deseleziona / Annulla filtro | Svuota selezione o chiude filtro |
| **Ctrl+Shift+E** | Esporta Excel | Esporta risultati correnti in Excel |

## 9. NOTE IMPLEMENTATIVE

### 9.1 Priorità Shortcut
- Le shortcut KBM NON devono sovrascrivere shortcut OS critiche (Ctrl+Z Undo, Ctrl+X/C/V Clipboard)
- Ctrl+Z: Undo dell'ultima modifica nel form (dove applicabile)
- Ctrl+Y: Redo (dove applicabile)

### 9.2 Conflitti Browser (se app web in RDP)
- Evitare Ctrl+W (chiude tab browser) — non mappare
- Evitare Ctrl+T (nuova tab) — non mappare
- Evitare Ctrl+R (reload browser) — preferire F5 per refresh app

### 9.3 Shortcut per Utenti Storici ERP
- **F3**: alias per Ctrl+N (legacy Zucchetti) — da valutare
- **F10**: alias per Ctrl+S (legacy Zucchetti) — da valutare
- **F8**: azione contestuale ("Conferma documento") — da definire per ogni modulo

### 9.4 Configurabilità
- Futuro (v2): pannello configurazione shortcut per utente
- MVP: shortcut fisse, documentate nell'help contestuale
